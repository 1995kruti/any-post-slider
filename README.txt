== Description ==

== Any Post Slider ==

Any Post Slider provides you the shortcode which will generate the slider for the selected post type to anywhere in the site. You just need to select the post type and required options from the settings page and it will generate the shortcode. That shortcode will generate the Owl carousel.

== Installation ==

1. Upload `any-post-slider` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
3. Find `Any Post Slider` inside your Settings menu and add the shortcode.
